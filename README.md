# ZAX Util
## support SSR Miniprogram Browser side

## install

``` base
npm i zax-util -S
```

## build with rollup

``` base
npm run build
```

## use

``` javascript 
const zaxUtil = require('zax-util')
or
import zaxUtil from 'zax-util'
```


``` javascript
 let { zaxString, zaxArray, zaxObject, zaxRegex, zaxRegexForm, zaxTypes } = zaxUtil
```
